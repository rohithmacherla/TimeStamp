Time Stamp API

Purpose: You can use this API to find the unix time stamp from a natural language time format or find the natural language time format from a unix time stamp. Learn about the unix timestamp.

Example input:

https://cryptic-brushlands-28381.herokuapp.com/December 4 2015
https://cryptic-brushlands-28381.herokuapp.com/1449187200

Example JSON output:

{"unix":1417651200,"natural":"December 4 2014"}

Example API call:

https://cryptic-brushlands-28381.herokuapp.com/October 4 1974
